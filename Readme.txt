This_Repository

This repository contains all the code for the given assignment.

    Through Live-SASS compiler on vscode was used to compile the css file.
    main.sass being the root file and all other components are named as such in sass folder.
    index.html is the starting point of this assignment followed by product.html.
    js folder holds the two files which in turn play two roles as file names mentioned in the webpage.
    Timeline and Commmits of My code can be traced with help of this repo.

Working and Implementation

    HTML5 tags are used.
    em and px are metrics used in this project.
    Flexbox and css Grid are used as main for styling and formating with help of Modern SASS.
    currecy_checkdown.js is used to covert the $ -> ? when USA to INDIA is selected.
    checkbox.js is used to display the items that belong only to that category.
    Media Queries are implemented to attain the desired responsive layout at two or three breaking points.

Testing

Webpage is developed Desktop-First Layout and testing is successful with proper display of images and colors.
Mobile Responsive is assigned to 470px for mobile view when converted to em=29.375em and Successful implementation.
